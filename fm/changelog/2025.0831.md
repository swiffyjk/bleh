hey this is the overdue monthly release for bleh – which!! has been getting a lot of attention recently so thank u for the love @w@

# Notice
- **If you didn’t see the warning or accidentally dismissed it when installing this update, please check you don’t have two instances of bleh installed. A weird quirk with how these extensions work is causing this version to act as a completely new script, so double-check!! >-<**

# Profiles
- __Introduced customisable profile accents for sponsors!__
	- A new ‘thank you :3’ reward for the sponsors of bleh, head to your user settings and you can pick a colour to brand your profile visible to every user
- __Extended markdown support to allow cool things__
	- Check out the new in-built markdown guide inside bleh to learn the basics – it’s found right below your about me box when editing!!
	- Try out `[left]`, `[right]`, and `[center` in your profile bio to align text to the way you desire (you need to also end with eg. `[/left]` so like `[center]hai im in the middle[/center]`).
	- There’s a new way to set a profile banner too! `[banner=url]` is the new recommended way to set your banner instead of the previous.
	- You can now also add formatted links to your socials! Wrap literal urls or formatted links with text inside a block of `[links] . . . [/links]` to have them display in a nice list with icons.
- __Extended the height of profile banners to be more unique__
	- Take over the entire top half of your profile to make your profile more your own!!
	- The ability to override profile banners with avatars now also only applies to **non-custom** images, aka. only ones sourced from the artist.
- __Re-arranged profile sidebars so you can see obsessions better__
- __Re-designed the way your taste compatibility is shown__
	- Very glowy now ~w~
- ‘aka.’ and ‘created’ for profiles are now cached so they display on subpages
- Changed design of badges to show text without hovering
- Redesigned event calendar icons to be cuter
- Finished migration of profile badges to support new version
	- Icons and colours can now also be loaded from the server to allow for seamless changes upon request
- __Added warning when clicking onto new links for the first time__
	- This is basically the norm in every single app now, if you click onto a new hostname (eg. google.com) for the first time, it’ll show a dialog asking if you really want to go there – along with a checkbox to hide it for next time you visit

# Friends
- __Re-introduced the ‘profile shortcut’ feature as an extension of the new ‘Friends’ feature__
	- Head to your bleh settings in the Profile tab and you can now add a list of your friends. In the future, this list will be used for showing what your friends are currently listening to and being able to see their individual plays on artists, albums, and tracks with a single click. For now, you can set your ‘profile shortcut’ using the new ‘starred friend’ option, which works the same but is now based on your bleh friends list.

# Music
- __Introduced a new default to ignore artist redirections__
	- Following the recent Last.fm updates to finally begin removing legacy redirections for artists like Travis Scott, bleh now automatically appends `+noredirect` to links you click to allow you to access the page you are most likely after. If this isn’t the case, you can disable in the ‘Music’ tab in Settings.
	- Along with this, ensure redirects are disabled in the *Last.fm ‘Website’ settings*
	- Page redirection notifications are now also disabled by default
- __The song tag library has expanded massively__
	- Thanks to pull requests primarily from @swiffyjk and @fr0r
- Fixed strange bulk edit conflict resulting in double album text
- __Fixed page toolbars being at the bottom in Chrome-based browsers__
	- Strange chrome incompatibility, worked just fine in Firefox
- __Introduced a new collage option for centered information__
- Tweaked the design of the social links to be more inviting
- Try out the new ‘Copy’ link when right-clicking tracks
- When navigating to the artist or album from the right-click menu, the links it uses should now be more accurate in some edge cases
- Fixed recent tracks on your homepage not being patched properly

# Inbox
- __Redesigned notification design from scratch__
	- Now fully built by bleh which took like forever to create and gather info for to work correctly but now it works and should be much clearer!!
- __Merged your notifications and mails together into one interface__
	- Next to your profile in the top-right you will now see a combined inbox button, on hover showing the counter for notifications and messages separately. On click opens a popup window showing your 5 most recent notifications and messages without having to swap pages.

# Navigation

	- Head to the ‘Profiles’ tab in settings to customise what links you use the most and which you don’t need – this includes bringing back individual links to notifications and messages if those interest you!
- __There is now a nice … menu in the top bar to access bleh-related options__
	- Helps if you are in need of support for any issues, want to suggest feedback, want to quickly add a new lotus capitalisation correction, or to sponsor <3
