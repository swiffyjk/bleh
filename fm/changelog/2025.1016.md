# oracle
- __Numerous improvements to the oracle test are now bundled into this release__
- __Cover arts are now loaded from oracle/hoshino sources if possible__
	- When browsing album pages and viewing album grids, the artwork url is now cached and used if needed.
	- When you encounter a track that has been cached by oracle (in a tracklist) or are on the track page, the artwork is now replaced to fit the album that was found.
- Blank disambiguation marker is now preferred over a ‘clean’ entry
- All featured artists are now checked, improving accuracy of feature detection
- Corrected weird usages of dashes and apostrophes
- Added `@connect` url to prevent options
- Tweaked design of music labels
- There may be cache issues (like overflowing) when using oracle, you can manage your cache in the ‘playback’ tab and clear away entries to save space for now

# Music
- __Fixed date range and various popups not closing as expected__
- Fixed links created by bleh looking different in encoding to Last.fm
	- Should fix difficulties in accessing artists with `+` in their name and things like that
- Fixed auto correct scrobbles option
	- Also now patched the playback preference option

# Design
- __Refreshed the halloween seasonal decorations__
- __Improved mobile navigation support!!__
- __Improved readability of grid text__
	- Also reduced the gap in between each grid item to prioritise artwork
- Redesigned radio buttons to match checkboxes
- You can now colour tracks in all places based on album cover
- Fixed the ‘Purple’ preset being orange during halloween
- Added an optional noise/grain overlay like before
- Usernames are now coloured based on ur badge in more places again
- Broadened usage of ‘solarium’ glass filter
- Tweaked design of buttons, links, playcount bars, music links, and more
- Entering a previously ‘invalid’ banner (not a url) will now simply use your accent colour
- Adjusted animations for tooltips and modals
