after 2-3 weeks the next version of bleh is now available :3 this update features a lot of polish, some new tools, and a LOT of work in the background.

# Collage
- __You can now generate 'collages' (aka. 'topsters', 'charts') on profiles__
    - Choose a timeframe, a type (albums, artists, tracks), a profile, a size (default is 5x5, maximum is 20x20), have a look through the options, then generate a collage image for download
- You can no longer start a collage or comparison when the profile has zero scrobbles

# Profile menu
- __Redesigned the profile menu__
    - The menu is now split into two sides with your avatar, name, badges, and shortcut on the left and your actions on the right split into pages. Same functionality, refined design.

# Design
- __Standardised a rounded, bordered design for all of bleh__
    - No more silly back-and-forth design changes like this from now on
- __Redesigned iconset__
    - New source for the icons, much cuter imo
- Updated design of user list in 'List' mode

# Structure
- __bleh now attempts to catch errors as early as possible__
    - Fingers-crossed this helps the weird 'black screen' issue people are encountering.
- __bleh has been 're-built' to use [lighterhtml](https://github.com/WebReflection/lighterhtml)__
    - This allows much easier development overall, majorly contributed by @yourfriendoss and the rest by me
    - There may be minor broken things due to this, those will be fixed most likely by the next update.
- __bleh will no longer crash if it can't load a profile badge__

# Notifications
- __Redesign notifications__
    - Notifications can now contain buttons and be paused on hover

# Settings
- __Fixed 'Avatar' colour option being incorrect sometimes__
- __Introduced new 'v2' settings components__
    - You will begin to see new text boxes (which support showing errors and submitting on pressing enter), menu buttons, and toggles

# Gallery
- You can now share gallery links

# Translations
- Translated to Português by @ArthRMH

# Other

- Artist and album titles are now corrected in your Bookmarks and the Releases page
- Fixed broken deleted shout icon and menu positioning
- Fixed artist activities having a broken tooltip
- Made side actions colourful on hover
- Made 'New' badges more subtle
- Fixed coloured shadow under no-cover grid items
- Added jsdoc to a few functions
- The sponsor list is now only updated if a higher version
- Fixed blank obsessor avatar
