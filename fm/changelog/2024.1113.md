# Redesigns
- __Completely redesigned headers for events, albums, tracks, artists, and profiles__
    - This update is crucial in removing legacy bleh leftovers. As of now, these headers are fully custom-built and now move away from the precisely positioned header and navigation seen before. In simple terms this makes the code much more straightforward and allow more flexibility to be possible with these designs in the future.

![Artist page](https://katelyynn.github.io/bleh/fm/changelog/img/artist.png)

- __Event pages have finally been redesigned__
    - Following with the header changes, event pages have seen a much needed refresh - go try it out :3
    - Lists of events on profiles and artist pages have also taken on this refresh.
    - Attendance pages now also include the view selector.
    - Some final tweaks may be made in the coming months or generally overtime.

![Event page](https://katelyynn.github.io/bleh/fm/changelog/img/event.png)

- The top-right profile menu is now custom-built
    - You may not notice a difference which is intentional, this simply improves performance drastically especially on old tabs.

- __The About artist panel has been redesigned__
    - Along with an updated design, this panel can now also display guest features on a song if enabled in settings.

![About artist panel](https://katelyynn.github.io/bleh/fm/changelog/img/about.png)

# Profiles
- __You can now right-click the similar taste item to compare artists__

![Taste right-click menu](https://katelyynn.github.io/bleh/fm/changelog/img/taste.png)

- You can now right-click tracks to view the menu
    - This will be used on artist/album grids and more in the future.
    - For those who aren't too happy with it, an option will be available later on to disable.

![Track right-click menu](https://katelyynn.github.io/bleh/fm/changelog/img/menu.png)

- __Configure menus now use the redesigned settings interface__
    - Very pretty :3

![Recent tracks menu](https://katelyynn.github.io/bleh/fm/changelog/img/recent-menu.png)

- You can now see badges in user lists (eg. following list)
    - This also means you can see staff users more easily.

- Auto-edit windows now use bleh toggles as appose to checkboxes
    - Along with this there are some very minor spacing fixes.

# Customisation
- __You can now control background saturation__
    - The saturation of backgrounds in dark themes can now be further controlled from the accent saturation using the new background saturation slider.
    - You can see this in action earlier in this changelog :3

![Background saturation slider](https://katelyynn.github.io/bleh/fm/changelog/img/saturation.png)

# Seasonal
- Season timings are now standardised solely in GMT+0000
    - While this may sound like a bad thing, this ensures there's no gap between one season ending and the other beginning (hopefully). Worth it in the end.

- Seasonal effects have had their (hopefully) final tweaks for Fall
- You can now decrease the amount of snow if needed for performance or accessibility
