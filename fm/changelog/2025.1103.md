hello all!! this is the latest release of bleh, originally meant for the end of October but plans changed @w@

# Performance
- __Fixed major performance regression from last releases__
	- bleh should now feel as smooth as ever again :3

# Sponsors
- __You can now change your ‘display name’ by sponsoring!__
	- Within your profile settings, you can set a custom ‘display name’ that shows only on your profile replacing your username (your username is then shown below)
	- This is also the reason usernames are now shown in most places with an @ at the start for clarity.
	- Have a look at my profile if you’re confused, @clairedoll
- __You can now choose a custom name style by sponsoring!__
	- Within your profile settings, you can pick from a range of fonts along with a few styling options (text shadow, glow) to show only on your profile
	- Have a look at my profile if you’re confused, @clairedoll

# Tracklists
- __Redesigned track layout customisation__
	- You now have full control over how tracklist items are displayed, with the following options at the moment:
	- **Track layout** controls if the track and artist name are aligned vertically or horizontally (like normal Last.fm)
	- **Show associated album name for tracks** gives you control of when the album name is shown, which by default is only when actively scrobbling
	- **Album name location** lets you decide (in vertical mode only) if the album name should be below the artist or to the side

# oracle
- __Refined how albums are prioritised__
	- The most ideal item to be selected is now the earliest by date that is an album and matches the title length the closest. Seems to be working much better when visiting a few pages than before.
- __Fixed guest and inline artists not showing anymore__
- __Added a dropdown selector to combine oracle tracklists, ‘Own plays’ tracklists, and Last.fm tracklists__
- Adjusted how cache is saved for the meantime
	- Album data is now no longer cached and cache time is now much shorter, this means album pages will make a request when you visit and tracks will probably lose their cover quicker in tracklists. Still a work in progress.
	- All track cache is now cleared out automatically without visiting each specific page.
